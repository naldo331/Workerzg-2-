# **App Name**: Workers Guild

## Core Features:

- Customer Job Request Form: Customers can post job requests with essential details (name, phone, location, category, description, budget, date/time), which are then saved for admin processing.
- AI-Powered Category Suggestion Tool: An AI tool assists customers by analyzing their job description input and suggesting the most relevant job categories, improving the accuracy of postings.
- Worker Registration & Profile Creation: Workers can submit applications including personal details, skills, experience, and preferred job categories. Upon admin approval, a basic worker profile is generated.
- Admin Job & Worker Management Interface: A streamlined admin dashboard to manually view all submitted job requests, review and approve/reject worker applications, and assign jobs to suitable workers.
- Worker Profile Display with Ranking: Viewable profiles for approved workers, showcasing their name, profile image, skills, and current Guild rank (Bronze, Silver, Gold, Platinum, Elite).
- Basic Job Communication Facilitation: A simple system to facilitate initial contact and communication between customers, workers, or admins, or to provide manual contact details for job coordination.
- Scalable Category Management: Admins have the capability to easily define, add, and manage the list of job categories to keep the platform adaptable to new service offerings.

## Style Guidelines:

- The primary color is a deep, rich indigo-plum (#563399), evoking a sense of professionalism, depth, and strength that aligns with a 'Guild' theme.
- The background color is a very dark, subtly purplish-grey (#1F1826), providing a premium, elegant dark theme that emphasizes content.
- The accent color is a brighter, energetic royal blue (#4D66EB), analogous to the primary and used to highlight interactive elements, calls-to-action, and key information.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and highly readable characteristics, suitable for a mobile-first, professional interface across all text elements.
- Utilize a set of clean, modern line icons that complement the contemporary UI and typography, ensuring clarity and intuitive navigation.
- A mobile-first, responsive design with clear content hierarchy, generous whitespace even within the dark theme, and intuitive form layouts for an optimal user experience on any device.
- Incorporate subtle and purposeful micro-interactions, such as elegant fades for content transitions and understated button feedback, to enhance user engagement without distraction.
